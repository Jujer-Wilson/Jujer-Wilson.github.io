1. Check Your JAVA_HOME Environment Variable#

Updating the Java_Home environment variable is an important step in getting your Android Studio working on Windows 10. The environment variable contains the path of Java and needs to be updated whenever a new version of Java is installed. Here’s how to update the Java_Home environment variable in Windows 10. Firstly, open the Windows “System” properties window by searching it in the start menu or press Windows+Pause. Next, click the Advanced system settings link and select the Environment Variables button. Then, in the “System variables” section, select “Edit” on the JAVA_HOME variable and the Edit System Variable window will appear. Finally, enter the correct path of the Java installation folder and click the “OK” buttons to save the settings. That’s it! You have successfully updated the JAVA_HOME environment variable in Windows 10.
